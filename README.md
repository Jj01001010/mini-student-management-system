# Mini-Student-Management-System

- This is a personal project of me using HTML, CSS, JS, Sweet Alert2, Font Awesome Icons & Google font API
- This project mainly focused for enhancing my skill on CRUD Operation

## Features

- Add students
- Edit students
- Delete students
- Display student list
- Show most common course
- Simple auth with pre-defined data for validations
